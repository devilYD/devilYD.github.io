# Vue学习笔记

## 开发环境搭建

### 安装nvm用于管理node.js版本

在Mac上使用brew直接安装即可

```bash
brew install npm
```

其他系统则可以参照[github仓库readme文件](https://github.com/nvm-sh/nvm)中的使用教程进行安装
